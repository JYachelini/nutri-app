# Nutritional Planner — Index

Weekly meal planning app, personalized for the user's nutritional goals. Stack: React (Vite) + TailwindCSS + localStorage.

Read the relevant file based on the task at hand.

## Visual work

For any task involving visual design — colors, layout, typography, new components, or styling decisions — use the `ui-ux-pro-max` and `frontend-design` skills. They can redefine the color palette from `docs/design.md`. Logic-only changes (balance calculation, data model, shopping list) do not require these skills.

## Starting point

`index.html` in the root is the fully functional prototype. All logic and UI is already solved there — use it as reference when porting to Svelte.
